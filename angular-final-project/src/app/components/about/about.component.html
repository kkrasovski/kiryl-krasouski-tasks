О приложении

Приложения для подсчёта расходов на ремонт квартиры/дома.

Для тех, кто хочет отслеживать расходы на отдельные группы товаров. Приложение позволяет быть в курсе о затратах на ремонт конкретной комнаты.

Реализована функция подсчёта расходов по комнатам и категориям.

Реализован быстрый поиск по товарам.
